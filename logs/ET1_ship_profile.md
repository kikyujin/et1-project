# 🚀 Elmar-Tail One - Ship Profile

## 基本データ
- **艦名**: Elmar-Tail One（エルマー・テイル・ワン）
- **艦種**: 筆型ワープ探査艇（Light-class Explorer）
- **形状/デザインモチーフ**: 万年筆
- **乗員**: マスター1名 + AI（エルマー他）

## 推進システム

### 主機関: 重力点穿孔ドライブ（Gravity-Point Puncture Drive）

- **基本原理**: マイクロブラックホール（極小特異点）を生成し、事象の地平線へ「落下」し続けることで推進
- **生成方式**: 
  - 船体周囲から磁場加速器（リング状）を展開
  - ペン先（ノーズコーン）の粒子加速器から重粒子（ヒッグス粒子群）を亜光速で射出・衝突
  - 衝突点に特異点（カー・ブラックホール）が誕生

### 航行エフェクト
- **特異点**: 「黒いインクの一滴」のように見える
- **周囲の光**: チェレンコフ光／ホーキング放射が青白く輝く
- **軌跡**: 「インクのような星屑」が舞う
- **ホログラム星図**: 紙にインクが滲むようにルートを描く

## 特殊機能

### E.V.H.S.（Elmar Visual HoloSystem）
- ホログラム星図の表示
- 空間演出UI
- 艦橋中央のローテーブルに配置

### 磁場加速器（Magnetic Field Accelerator）
- ワープ時に船体周囲（万年筆の胴軸部分）に展開
- リング状の幾何学模様を描く

## 航行シークエンス（ワープ手順）

```
1. 「重力場、展開」
   → 磁場加速器が幾何学模様を描いて展開

2. 「ターゲット座標固定。インク、ロード」
   → ヒッグス粒子充填

3. 粒子ビーム射出
   → ペン先から放たれたビームが一点で交錯
   → 衝突点に特異点（黒点）生成

4. ダイブ
   → 船が自ら作った奈落へ落下——ワープ開始
```

### シークエンス描写例

> **マスター**: 「重力場、展開。ターゲット座標固定。インク、ロード。」
>
> **エルマー**: 「磁場加速器、展開完了！ ヒッグス粒子充填率120%！ いつでもイケるよ、にーに！」
