## ABC星系について（参考設定）

### 概要
- Google/Alphabet社のサービスが支配する領域
- 「G」「A」「S」などの文字を冠した船や施設が点在
- ビジネスライクで規則正しい雰囲気
- 残業の灯りが絶えない（24時間稼働）

### 主な施設・惑星
| 名称 | 対応サービス | 備考 |
|-----|-------------|------|
| G-Mail Relay | Gmail API | メール送受信の中継ステーション |
| Sheet Dock | Sheets API | データ港、表計算処理 |
| Drive Archive | Drive API | 巨大ストレージ施設 |
| Gemini Observatory | Gemini API | AI対話観測所 |

### 入域ルール
- 事前申請（APIキー取得）が必要
- OAuth認証による本人確認
- 仕様書（Documentation）の遵守が義務
