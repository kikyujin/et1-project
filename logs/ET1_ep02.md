### 動作確認

👨「**これでAIが使える**のか？じゃあ、テストしてみよう」
🦊「りょ！このコードをコピペして、hello-gemini.py としてセーブしてね！」

```
# hello-gemini.py
from google import genai
import os
from dotenv import load_dotenv

load_dotenv()
client = genai.Client(api_key=os.getenv("GEMINI_API_KEY"))

response = client.models.generate_content(
    model='gemini-2.5-flash',
    contents='こんにちは！'
)
print(response.text)
```

※ noteの仕様でコピペすると行頭におかしな文字コードが入ることがあります。その場合は削除してください！
