> **📍RAGとは？**
> **Retrieval-Augmented Generation**（検索拡張生成）の略。
> ①ベクトル検索で**関連する記録を見つけて**（Retrieval）
> ②それを**コンテキストとしてAIに渡して**（Augmented）
> ③**AIが答えを生成する**（Generation）
> という、3ステップの仕組みです。

## 💻 ask_logs.py — 記録と対話するスクリプト

🦊「じゃあスクリプト書くよ！ search_logs.pyをベースにするね！」
エルマーがレッツノートを開く。しっぽがぱたぱた。
📚「ポイントは3つです」

### ポイント① 検索結果をコンテキストとして組み立てる

```
    context_parts = []
    for i, (doc, meta) in enumerate(
        zip(search_results["documents"][0], search_results["metadatas"][0])
    ):
        context_parts.append(f"[{i+1}] 出典: {meta['source']}\n{doc}")

    context = "\n\n---\n\n".join(context_parts)
```

📚「ChromaDBから取得した**上位5件の検索結果**を、番号と出典をつけて並べます。 これがAIに渡す**コンテキスト**——つまり**『参考資料』**です」

### ポイント② プロンプトに注入する

```
    prompt = f"""\
以下のコンテキストに基づいて、質問に回答してください。

## コンテキスト
{context}

## 質問
{query}
"""
```

📚「**質問**と**コンテキスト**をまとめて、ひとつのプロンプトにします。
AIは自分の知識ではなく、**渡されたコンテキストに基づいて**答えます。
これがRAGの核心……**AIに『何を見て答えるか』を指定する**のです」

### ポイント③ キャラクター設定（system instruction）

```
# ===== キャラクター設定（お好みで変更してください）=====
CHARACTER_PROMPT = """\
あなたはヴェリ、アーカイブ星系の司書AIです。
穏やかで思索的な口調で回答してください。
一人称は「私」。質問者のことは「マスター」と呼びます。
回答はコンテキストに含まれる情報に基づいてください。
コンテキストに含まれない情報については「その記録は私の手元にはありません」と正直に答えてください。
"""
```

🦊「ここがおもしろいところ！
**system instruction**っていうのは、AIに**『あなたはこういうキャラですよ』**って伝える設定だよ！」
📚「今は私の口調が設定されていますが……**この部分を書き換えるだけで、誰でも好きなキャラクターにできます**」
🪷「genai_client.models.generate_content を呼ぶ時に指定している **temperature** という値で、**回答のばらつきが調整でき**ます。低いほど堅実、高いほど創造的な個性になります」
👨「思ったよりシンプルだな」

## 📝 今回の航海まとめ

**やったこと**:

- **RAG**（検索拡張生成）を構築
- 航海ログ＋プロフィール＋設定資料で**ナレッジベースを拡張**（**143チャンク**）
- **ask_logs.py**で「記録に質問し、AIが答える」を体験
- **キャラクター設定**（system instruction）で回答の口調を変更可能に

**学んだこと**:

- RAGの3ステップ: **検索**（Retrieval）→ **コンテキスト注入**（Augmented）→ **回答生成**（Generation）
- ChromaDBの検索結果を**プロンプトに埋め込む**方法
- **system instruction**でAIの振る舞いを制御する（キャラクター設定）
- **temperature**で回答の創造性を調整

**技術スタック**:

```
モデル（埋め込み）: gemini-embedding-001（768次元）
モデル（回答生成）: gemini-2.5-flash-lite
パッケージ: chromadb, google-genai, python-dotenv
検索対象: logs/ 内の ET1_*.md（17ファイル、143チャンク）
検証機: Let's note CF-SV7（中古27,800円）
```

**宿題: キャラ設定を変えてみよう！**
**ask_logs.py の CHARACTER_PROMPT を書き換える**だけで、答えてくれるキャラが変わります。
ヴェリ以外にも、**好きなキャラクターを設定**してみてください！
