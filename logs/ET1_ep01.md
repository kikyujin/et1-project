## 📦 仮想環境（venv）を作る

🖤「次、**仮想環境**を作るよ。**venv**っていうの」
👨「仮想環境？」
🖤「**プロジェクトごとにPythonの環境を分ける**の。
システムのPythonを汚さないためね
……あ、その前に。venvのパッケージを入れておいて」

```
sudo apt install python3.12-venv
```

👨「sudoはパスワード必須」
🖤「なにか聞かれたら、"Y"でOKだから！」

```
mkdir ~/et1-project
cd ~/et1-project
python3 -m venv .venv
```

👨「できた！」
🖤「使う時は**仮想環境の有効化をする**のよ」

```
source .venv/bin/activate
(.venv) master@ElmarTailOne:~/et1-project$
```

🖤「**プロンプトに (.venv) がついた**でしょ。
これが仮想環境の中にいるっていうこと」
👨「なるほど」
🖤「作業が終わったら **deactivate** で**仮想環境から抜けられる**ので覚えておいてね！」
