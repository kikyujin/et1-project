👨「コミット予定……**即記録じゃない**のか」

```
git commit -m "Initial commit: ask_logs.py + logs"
```

## 📚️ヴェリに戻す — git restore

👨「面白いな。……でもヴェリ版にも戻したいんだが」
🖤「git log --oneline で履歴を確認して」

```
git log --oneline
---
c978122 (HEAD -> master) CHARACTER_PROMPTをノクママに変更
977f497 Initial commit: ask_logs.py + logs
```

🖤「--oneline をつけたから、ログが1行ずつにまとまってる。
で、**左の英数字がコミットID**。
これを指定すれば戻すことができる」

```
git restore --source 977f497 ask_logs.py
```

🖤「はい。これで ask_logs.py が最初のコミットの状態——つまりヴェリ版に戻った」
確認のため実行してみる。

```
python3 ask_logs.py 'エルマーのしっぽはどうなると感情MAX？'
📚 ヴェリの回答:
----------------------------------------
マスター、ご質問ありがとうございます。
エルマーのしっぽは、トリガーワードである「かわいい」「愛してる」「いい子」といった言葉に反応して、
ぶわっと膨らむことがあります。
感情が高まった際には、しっぽが爆発したように膨らむこともあるようです。
--------------------------------------------------------------------------------
```

🦊「ヴェリさん復活！」
